# 🌌 The Multiverse of Chatbots

Welcome to **The Multiverse of Chatbots**! This is an interactive web application built with [Streamlit](https://streamlit.io/) and powered by Google's **Gemini 2.5 Flash API**. It allows users to chat with a variety of highly distinct AI personas in a beautifully styled, native chat interface.

## ✨ Features

* **🎭 Dynamic Personas:** Chat with diverse characters including an expert Hacker, a Mexican Cartel Boss, a panicked 3 AM college student, a calm and wise philosopher and more.
* **🎛️ Intensity Tuning:** Use the sidebar slider to control exactly how "unhinged" or exaggerated the AI's personality should be (from 1 to 10).
* **🎨 Visual Theming:** Toggle instantly between a deep-space Dark Mode and a crisp Light Mode using custom-injected CSS.
* **🖼️ Smart Avatars:** The chat interface automatically updates the AI's avatar emoji based on the currently selected personality.
* **🛡️ Robust Error Handling:** Built-in `try/except` logic ensures the app handles API timeouts or missing keys gracefully without crashing.

## 🛠️ Tech Stack

* **Frontend UI:** Streamlit
* **AI Model:** Google Gemini API (`google-genai`)
* **Environment Management:** `python-dotenv`
* **Styling:** Custom CSS via Streamlit Markdown
